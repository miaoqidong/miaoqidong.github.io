新版微信无法识别或识别错误的解决方案

由于微信最新的 8.0.55 、8.0.56 版本在对无障碍功能的支持上存在BUG或者故意为之,导致所有 无障碍相关的软件(如自动记账) 无法正常获取微信的界面数据 ,该问题不知道是故意为之还是无意的错误, 需等待微信自己修复

目前有两种解决方案

①安装使用 8.0.54 及之前版本的微信（可以通过微信电脑版备份和恢复聊天数据）微信历史版本或使用 adb install -r -d path/to/your/app.apk 强制降级安装，这样无需卸载和迁移数据

②如果不方便回退微信版本，可以在系统设置中搜索 无障碍，进入 无障碍-视觉-随选朗读，开启 随选朗读快捷方式,开启后可以让微信 部分界面(多少部分不确定) 的无障碍数据恢复正常

选择方案2开启 随选朗读快捷方式 后，如果屏幕边缘显示无障碍图标, 有两种方案可以隐藏这个图标

①到无障碍界面 "无障碍"按钮-使用按钮或手势 点击切换为手势即可隐藏

②将此快捷方式设置为 按住音量键
